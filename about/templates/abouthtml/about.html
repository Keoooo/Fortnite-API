The Fortnite API

 

What is this?

The Fortnite Skins API is a RESTFUL API based on the popular game Fortnite. You will have access to all the skins which have been released by epic games.

 

Who are you?

My name is Andrew Keogh I have recently started learning python and creating awesome projects.

 

Why did you build this?

I wanted to build a nice little portfolio and thought it would be fun.

 

Did you build this API alone?

Yes, but if you want to get involved contact me all help and advice is welcomed  

 

Inspiration

I took a lot of inspiration from The Rick And Morty API check out this awesome site!  

 

Technical stuff?

I am using Django to create the site and API. All the data is formatted in json and I'm hosting the entire project on Digital Ocean.
